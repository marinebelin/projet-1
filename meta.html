BALISE <META>

DESCRIPTION

Les balises <META> servent à placer des métadonnées (metadata) dans une page HTML. On placera ces informations dans l’élément head, et elles ne seront pas affichées sur la page. Elles sont destinées aux navigateurs web, aux moteurs de recherche; et plus largement, à tous les outils d’indexation, c’est-à-dire tous les outils qui analysent vos pages pour y identifier diverses informations.

ATTRIBUTS

ATTRIBUTS DESIGNATION UNITE ACCEPTEE
charset Permet de spécifier aux navigateurs le format d’encodage de vos pages web. Il est préférable d’utiliser le format UTF-8 qui supporte une grande variété de langages (même mélangés au sein d’une même page) et prend en compte les caractères spéciaux comme les accents. UTF-8
http-equiv="Content-Language" Il indique la ou les langues du site. content="fr"
name="keywords" Informe les moteurs du contenu du site avec des mots-clés* (maximum 1000 caractères). content= "ici les mots clés séparés par des virgules"
name="Description" Cet attribut indique la description du site (maximum 200 caractères). content = "ici la description du site."
name="Copyright" Permet de poser un copyright. Content = "ici le nom du copyright"

EXEMPLE

<head>

<title>META</title>

</head>


<body>

<p> Un exemple de code pour montrer l'utilité de la balise meta. Les accents comme é, è, à,... ainsi que les caractères spéciaux comme &, $, €,... ne s'affichent pas correctement sans la balise.</p>

</body>


<head>

<meta charset="utf-8"/>

<title>META</title>

</head>
