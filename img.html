BALISE <IMG>

DESCRIPTION

La balise <IMG> permet de mettre une image dans le document html.

ATTRIBUTS

ATTRIBUTS DESIGNATION UNITE ACCEPTEE
class On se sert souvent des noms de classes pour appliquer des styles. Liste de caractères
id Cet attribut permet d'assigner un nom à l'instance d'élément où il apparaît. Il peut être utilisé comme destination pour un hyperlien, référence pour un script, identifier une déclaration de l'élément object, ou encore permettre aux feuille de styles d'appliquer un style à un élément en particulier. Liste de caractères
Longdesc l'attribut Longdesc précise un lien vers une description longue de l'image. Url
Name Cet attribut permet d'identifier la balise Liste de caractères
Src L'attribut Src spécifie l'URL de l'image à charger. Url
Style L'attribut Style permet de définir des cascading style sheets dans la balise même. Css
Title L'attribut Title permet d'afficher un petit texte comme sur les liens à côté du mot lorsque la souris est dessus. Liste de caractères
Usemap L'attribut Usemap permet d'associer l'image à une Map (zone réactive pour les liens). Liste de caractères
Width L'attribut Width spécifie la longueur de l'image px ou pourcentage
Eight L'attribut Width spécifie la hauteur de l'image px ou pourcentage
