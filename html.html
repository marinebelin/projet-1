BALISE <HTML>

DESCRIPTION

La balise <HTML> permet la déclaration du document HTML. On lui rajoutera l'attribut lang pour déclarer la langue utilisée.

ATTRIBUTS

ATTRIBUTS DESIGNATION UNITE ACCEPTEE
acesskey Il fournit une indication afin de générer un raccourci clavier pour l'élément courant. Liste de caractères
class On se sert souvent des noms de classes pour appliquer des styles. Liste de caractères
contenteditable Il permet de préciser si le contenu de l'élément peut être édité ou non. Il s'agit d'une fonctionnalité répondant à la demande du web participatif : tout un chacun doit pouvoir éditer en ligne du contenu, le mettre en forme, sans posséder de grandes connaissances en HTML. Booléen
id Cet attribut permet d'assigner un nom à l'instance d'élément où il apparaît. Il peut être utilisé comme destination pour un hyperlien, référence pour un script, identifier une déclaration de l'élément object, ou encore permettre aux feuille de styles d'appliquer un style à un élément en particulier. Liste de caractères
dir est un attribut à valeur contrainte qui indique la direction du texte contenu dans l'élément. Ltr, rtl ou auto

EXEMPLE

<!doctype html>

<html>

<head>

<meta charset="utf-8"/>

<title>Description</title

</head>


<body>

<p>Bonjour et bienvenue sur mon site.</p>

</body>

</html>
