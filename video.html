BALISE <VIDEO>

DESCRIPTION

La balise <VIDEO> est une toute nouvelle addition au HTML5 et vous permet, vous le devinerez, de visionner un film sur votre site Web. Les données reliées à cette balise sont normalement des vidéos, mais peuvent aussi inclure des images ou des données sonores.

ATTRIBUTS

ATTRIBUTS DESIGNATION UNITE ACCEPTEE
src L'attribut qui permet d'entrer l'adresse du fichier vidéo. URL
type Il est conseillé de mettre plusieurs formats différents en indiquant l'extension de chaque source grâce à l'attribut type. video/mp4 ; video/webm ; video/ogg ;...
autoplay Un attribut qui vous permet d’activer le vidéo dès que la page est téléchargée. Booléen
poster permet d'indiquer une image à afficher par défaut dans l'espace réservé par la vidéo, avant que la lecture de celle-ci ne soit lancée. URL
controls L'attribut controls donne accès aux contrôles de lecture (boutons de navigation, volume, etc, selon les possibilités du navigateur), ou les masque s'il est omis. controls="controls"
loop Cet attribut indique que la vidéo doit être jouée en boucle. Booléen
width La largeur de la zone où afficher la vidéo. px
height La hauteur de la zone où afficher la vidéo. px

EXEMPLE

<video width="400" height="222" controls="controls">

<source src="video.mp4" type="video/mp4"/>

<source src="video.webm" type="video/webm"/>

<source src="video.ogv" type="video/ogg"/>

</video>
