BALISE <STRONG>

DESCRIPTION

La balise <STRONG> indique une mise en exergue du texte, en la mettant généralement en gras.

ATTRIBUTS

ATTRIBUTS DESIGNATION UNITE ACCEPTEE
class On se sert souvent des noms de classes pour appliquer des styles. Liste de caractères
dir Cet attribut permet de donner une direction de lecture au texte. LTR ou RTL
id Cet attribut permet de spécifier la classe css à utiliser sur la balise dans les cas des feuilles de style. Dans le cas du Javascript, il permet de donner l'identifiant pour accéder aux propriétés de la balise strong Liste de caractères

EXEMPLE

<STRONG> </STRONG>
